Supabase Config Checklist ⚡

I couldn't bypass the login screen to check automatically, so please verify these 3 points in your Supabase Dashboard.

1

Facebook Provider Enabled

Go to Authentication -> Providers -> Facebook. Ensure the toggle is ON.

2

App Secret Verification

Double-check that the App Secret matches the one in your Facebook Dashboard (Settings -> Basic).

3

Client ID (App ID)

Ensure the Client ID is exactly 26587514167520216.

Why the Key Hash is still the #1 suspect

Even if Supabase is perfect, Facebook will block the connection before the user ever sees a login screen if the Key Hash doesn't match the one sending the request.

"L'application n'est pas configurée : Cette application n'a pas de configuration Android valide."

Common error message when Key Hash is wrong.
