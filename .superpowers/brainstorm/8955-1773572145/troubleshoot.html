Facebook Login Troubleshooting

Let's narrow down where the connection is breaking.

1

The button doesn't do anything

I click 'Continuer avec Facebook' and nothing happens.

2

Browser opens but shows an error

Facebook says "Invalid OAuth Redirect URI" or "App Not Setup".

3

Login succeeds but doesn't return to app

I log in successfully, but I stay in the browser or it opens a blank page.

4

Returns to app but I'm not logged in

The app reopens, but shows an error or stays on the login screen.

Current Configuration Check

What looks correct

  • AndroidManifest.xml Intent Filters
  • strings.xml (App ID & Tokens present)
  • AuthScreen implementation

Potential Culprits

  • Supabase Redirect URI whitelist
  • Facebook Dev Portal settings (Android Platform)
  • Key Hashes (Release vs Debug)
